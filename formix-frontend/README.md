# Formix Frontend

Aplicação frontend do Formix, construída com React e Next.js.

## Stack

- React
- Next.js (LTS)
- TypeScript

## Estrutura

```
src/
  app/          # Rotas e páginas (App Router)
  components/   # Componentes reutilizáveis de UI
  modules/      # Componentes específicos de domínio
  hooks/        # Custom hooks
  services/     # Comunicação com API
  styles/       # Estilos globais e temas
  types/        # Tipos TypeScript compartilhados
```

## Funcionalidades principais

- **FormBuilder** — Editor visual de formulários com múltiplos tipos de perguntas
- **Respondente** — Interface pública para responder formulários
- **Dashboard** — Visualização de analytics com gráficos e estatísticas
- **Gestão** — Painel administrativo para usuários, organização e convites
